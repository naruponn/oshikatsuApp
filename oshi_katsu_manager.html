⚡ Googleスプレッドシート連携設定
STEP 1 — Googleスプレッドシートを新規作成
▶ sheets.new を開くしてシートを作る。シート名は何でもOK。
STEP 2 — Apps Script を開く
メニュー「拡張機能」→「Apps Script」をクリック。
STEP 3 — 下のコードを貼り付けて保存（Ctrl+S）
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet(); var data = JSON.parse(e.postData.contents); var tab = sheet.getSheetByName(data.sheet) || sheet.insertSheet(data.sheet); if (tab.getLastRow() === 0) { tab.appendRow(data.headers); } tab.appendRow(data.row); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }
STEP 4 — デプロイ
右上「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」→
「アクセスできるユーザー」を 「全員」 に変更 → 「デプロイ」ボタン押す。
表示されたURLをコピー。
STEP 5 — URLを下に貼り付けて保存
